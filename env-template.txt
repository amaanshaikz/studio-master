# SETUP INSTRUCTIONS
# =============================================================================

# 1. SUPABASE SETUP:
#    - Go to https://supabase.com/
#    - Create a new project
#    - Copy the URL and anon key from Settings > API
#    - Copy the service role key from Settings > API (keep this secret!)

# 2. NEXTAUTH SETUP:
#    - Generate a secret: openssl rand -base64 32
#    - Set NEXTAUTH_URL to your domain (http://localhost:9002 for development)

# 3. GOOGLE AI SETUP:
#    - Go to https://makersuite.google.com/app/apikey
#    - Create an API key for Gemini API

# 4. VERTEX AI SETUP (Optional):
#    - Go to https://console.cloud.google.com/
#    - Create a project and enable Vertex AI API
#    - Set up billing (required for Vertex AI)
#    - Run: gcloud auth application-default login
#    - Run: gcloud services enable aiplatform.googleapis.com

# 5. INSTAGRAM SETUP:
#    - Go to https://developers.facebook.com/
#    - Create a new app
#    - Add Instagram Basic Display product
#    - Configure OAuth redirect URIs:
#      * http://localhost:9002/api/auth/callback/instagram
#      * https://yourdomain.com/api/auth/callback/instagram
#    - Set webhook URLs (optional):
#      * https://yourdomain.com/api/webhooks/instagram

# 6. LINKEDIN SETUP:
#    - Go to https://www.linkedin.com/developers/
#    - Create a new app
#    - Configure OAuth redirect URIs:
#      * http://localhost:9002/api/auth/callback/linkedin
#      * https://yourdomain.com/api/auth/callback/linkedin
#    - Set webhook URLs (optional):
#      * https://yourdomain.com/api/webhooks/linkedin

# 7. SECURITY SETUP:
#    - Generate encryption key: openssl rand -hex 16
#    - Generate webhook verify tokens (any random strings)
#    - Keep all secrets secure and never commit them to version control
